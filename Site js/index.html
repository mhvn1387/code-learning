variable and constant

let

Variables defined with let can not be redeclared.You can not accidentally redeclare a variable declared with let.

let x = "John Doe";
let x = 0;

var

Variables are containers for storing data (storing data values).In this example, x, y, and z, are variables, declared with the var keyword:

var x = 5;
var y = 6;
var z = x + y;

const

A const variable cannot be reassigned:

const PI = 3.141592653589793;
PI = 3.14;      // This will give an error
PI = PI + 10;   // This will also give an error
